# MoneroBot
Simple Telegram bot that lets you know Kraken's XMR-EUR change

## Dependencies
From NPM:
  * **kraken-api** by 5an1ty
  * **node-telegram-bot-api** by yagop

## How to use
* clone this repo
* `npm install`
* change *YOUR_KEY* in `main.js` with your Telegram bot token
* `node main.js`
